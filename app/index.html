О компании

Джи-Эс-Ти работает на самарском рынке ИТ-услуг с 2005 года. За годы работы компания накопила опыт реализации проектов в различных областях информационных технологий.Благодаря тесным партнерским отношениям с мировыми производителями высокотехнологичного оборудования и программного обеспечения, таких как: Hitachi Data System, IBM, Fujitsu, Kraftway, Lenovo, D-Link, Xerox, Canon, Konica-Minolta, Brother, Microsoft Corporation, компания имеет возможность предложить своим клиентам новейшие высокотехнологичные продукты и эффективные решения.

Джи-Эс-Ти работает для клиентов и вместе с ними. Каждый клиент компании – партнер по взаимовыгодному и долгосрочному сотрудничеству.

Индивидуальный подход к решению любых задач – один из важных принципов работы. Исходя из этого, компания Джи-Эс-Ти стремимся предоставлять максимально качественные услуги для развития и продвижения бизнеса.

Реализованные проекты

1 Оптимизация системы корпоративной электронной почты и внедрение Symantec Enterprise Vault для ОАО «Сибур Холдинг».
ЗАДАЧА Повышение удобства пользования корпоративной электронной почтой, возможность быстрого доступа к архивам за прошлые периоды, обеспечение бесперебойной работы корпоративной почты, исключение рисков потери и искажения данных.
РЕШЕНИЕ В рамках проекта было реализовано комплексное решение Symantec Enterprise Vault, предназначенное для архивирования сообщений, а также для архивирования файловых серверов.
РЕЗУЛЬТАТ Оптимизация корпоративной почтовой системы обеспечила: Централизованное хранение все возрастающих объемов электронной почты с автоматическим архивированием по заданным правилам, определяемым корпоративной политикой, что позволило пользователям использовать почтовый ящик практически неограниченного размера при сохранении контроля над ростом ресурсов.
